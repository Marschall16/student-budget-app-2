# Серверное API (ServerApi)

## Что находится в этой папке
Бэкенд-часть проекта: REST API сервер, модели данных, бизнес-логика.

## Планируемые модули
- Модуль аутентификации (JWT)
- Модуль пользователей (CRUD)
- Модуль ресторанов и меню
- Модуль заказов
- Модуль оплаты (интеграция с платежными системами)

## Взаимодействие
- Предоставляет API для MobileApp и Web.
- Работает с базой данных.
