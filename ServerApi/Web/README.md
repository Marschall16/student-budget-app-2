# Веб-сайт (Web)

## Что находится в этой папке
Код веб-версии приложения для доставки еды (SPA на React/Vue).

## Планируемые страницы
- Лендинг с презентацией сервиса
- Каталог блюд с фильтрацией
- Страница ресторана
- Корзина и оформление заказа
- Админ-панель

## Взаимодействие
- Отправляет запросы к ServerApi.
- Может использовать те же компоненты, что и MobileApp (дизайн-система).
