# Мобильное приложение (MobileApp)

## Что находится в этой папке
Исходный код клиентского мобильного приложения для iOS и Android.

## Планируемые компоненты
- Экран входа и регистрации
- Главный экран с каталогом блюд
- Экран деталей блюда
- Корзина и оформление заказа
- Личный кабинет и история заказов

## Взаимодействие с другими частями проекта
- Отправляет запросы к ServerApi для получения меню и оформления заказов.
- Использует WebView для отображения страниц оплаты (при необходимости).
